# ビルド修正 Skill

## 目的

ビルドエラーを段階的に解消し、新しい不具合の混入を防ぐ。

## 手順

1. ビルドを実行してエラーを収集する
2. エラーをファイル単位に整理する
3. 1件ずつ修正する
4. 各修正後にビルドを再実行する

## 停止条件

- 同じエラーが3回連続で解消しない
- 修正によって新規エラーが急増する
- ユーザーが一時停止を指示する

## 報告項目

- 解消済み件数
- 未解消件数
- 新規発生したエラー
