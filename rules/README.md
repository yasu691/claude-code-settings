# ルール一覧（共通）

このディレクトリは、Claude Code が常時参照する「汎用ルール」を置く場所です。
Hook には依存せず、文章ルールとして運用します。

## 収録ファイル

- `coding-style.md`：実装スタイルと可読性ルール
- `testing.md`：テスト方針とTDD運用
- `security.md`：セキュリティ最低基準
- `git-workflow.md`：コミット・PR運用

## 運用方針

- まずはこの4ファイルだけを維持する
- プロジェクト固有ルールは各リポジトリの `CLAUDE.md` 側に追加する
- ルールは短く、判断基準が明確な文章にする
